#Project 1: Board Games
#Summary: This Python application helps users find the perfect board game by parsing a custom dataset via the Pandas library. It features an interactive, loop-driven menu that takes user preferences and randomly recommends a matching game based on various filters.
#Key Features: 
- Recommends board games based on specific categories (e.g., adventure, fantasy), game mechanics, exact play times, or group sizes.
- Uses Pandas to pull and structure game data directly from an external CSV file.
- Populates a real-time list of matches based on user input and leverages the random module to pick a unique suggestion every time.

#Project 2: Dog Breed 
#Summary: This Python application assists users in finding a dog breed that matches their lifestyle and preferences by analyzing a dedicated dataset using Pandas. Through an interactive terminal menu, users can filter breeds by physical traits and behaviors to receive randomized recommendations.
#Key Features: 
- Recommends specific dog breeds based on user preferences for size classification (tiny, small, medium, large) or purposeful traits (e.g., herding, hunting, companionship).
- Includes a dedicated temperament lookup feature that outputs the breed's personality traits and automatically opens a web browser link to an image of the dog.
- Along with a main random recommendation, the script displays the full list of alternative matching breeds so users can explore all options.

#Project 3: Calculator 
#Summary: This Python script functions as a basic arithmetic calculator that guides users through essential mathematical operations. It prompts the user for two integers and a choice of operator, computes the result using dedicated mathematical functions, and prints the final output to the console.
#Key Features: 
- Separates computational logic from the main interface by isolating addition, subtraction, multiplication, and division into individual, reusable functions.
- Utilizes an if-elif-else control flow structure to map user-inputted operator symbols directly to their corresponding mathematical operations.
- Uses clear instruction prompts and organized console logs to ensure a smooth, conversational user experience.

#Project 4: Rock Paper Scissors 
#Summary: This Python program is an interactive simulation of the classic game "Rock, Paper, Scissors" where a player competes directly against the computer. The script loops continuously, processing the player's choices against random computer selections while dynamically tracking and printing the running scores for both participants.
#Key Features: 
- Utilizes global variables and continuous arithmetic updates to maintain and display an ongoing score match history between the player and the AI.
- Features an exhaustive conditional evaluation system that handles all nine possible game outcome permutations (wins, losses, and ties).
- Combines a while True main loop with an interactive end-of-round replay prompt, letting users easily play multiple rounds or break the loop to quit.

#Project 5: Pokémon
#Summary: This terminal-based RPG lets players raise a Squirtle through training and battles to increase its level and trigger classic Pokémon evolutions. The game operates on a daily loop structure where players balance risk and reward to prepare their Pokémon for a high-stakes final boss battle.
#Key Features: 
- Monitors player level thresholds to automatically evolve the Pokémon from Squirtle to Wartortle at level 10, and Blastoise at level 20.
- Features large-scale, detailed multi-line ASCII graphics that visually update the Pokémon's appearance on screen based on its current evolutionary stage.
- Implements a probabilistic boss fight where your chances of winning scale dynamically with your level, reducing the statistical likelihood of losing as your Pokémon grows stronger.
